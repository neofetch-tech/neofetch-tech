# Hi there, I'm Liquidarium! 👋

### 🛠️ About Me
I'm a web and game developer focused on building interactive web applications, high-performance 2D game engines, and immersive digital experiences. I love architecture design, clean code, and pushing the boundaries of what's possible in the browser and desktop apps.

- ⚡ **My Current Stack:** **JavaScript & TypeScript** + **React** for modern web interfaces, and **PixiJS** for high-performance 2D canvas rendering and custom game engine architecture.
- 🎯 **Next Frontier:** Mastering game loop optimization, state management in complex systems, and bundling web-based games into desktop applications.
- ⚙️ **Current Projects:** - **Custom 2D Game Engine** — A proprietary, data-driven 2D engine built from scratch using TypeScript and PixiJS.
  - **Undertale-inspired Indie Game** — Atmospheric, cozy, story-driven game with custom dialogue systems and retro aesthetics.
- 🚀 **Goal:** Becoming a top-tier Game & Frontend Engineer, releasing my own games on Steam, and building robust open-source tools for the developer community.
